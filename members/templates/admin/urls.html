  • Root script URL: {$root_url}/
    There's not index.php page! This URL is possible (but not recommended) root of your site and you should create index.html (or index.php) site manually if it needed.
  • Member Signup Page: {$root_url}/signup.php
  • Member's Page (for renewal, view payment history, buy other products): {$root_url}/member.php
  • PayPal IPN Page (for use in PayPal settings): {$root_url}/plugins/payment/paypal/ipn.php
  • ClickBank Thanks Page (for use in ClickBank settings): {$root_url}/plugins/payment/clickbank/thanks.php
  • Script Root Directory: {$config.root_dir}
  • .htaccess sample (cut and paste, just change product #)
    AuthType Basic
AuthName "Your site protected dir description - possible the same for entire site"
AuthUserFile {$config.root_dir}/data/.htpasswd
AuthGroupFile {$config.root_dir}/data/.htgroup
Require group PRODUCT_1 PRODUCT_3 ## if users of subscriptions 1,3 have access
